백틱 확장 파싱 테스트 결과물 (bt_examples.html)

이 문서는 bt_examples.md 마크다운 소스가 최종적으로 웹 화면에 렌더링되어야 하는 모습을 담은 정적 목표 파일입니다.

  1. 풀 포맷 (색상 + 아이콘 + 텍스트)
    노란색 배경에 편집 아이콘 등록 버튼:
    등록
  2. 색상 생략 (기본 색상 + 아이콘 + 텍스트)
    기본 배경(회색)에 플러스 아이콘 추가 버튼:
    추가
  3. 아이콘 생략 (색상 + 텍스트)
    붉은색 배경에 저장 텍스트:
    저장

    노란색 배경에 인사정보 가져오기 텍스트:
    인사정보 가져오기
  4. 색상, 아이콘 모두 생략 (사전 등록 단어 자동 룩업)
    기등록 단어 "등록" (초록색 2EAC7E 배경 + edit 아이콘):
    등록

    기등록 단어 "삭제" (붉은색 DD5E5E 배경 + trash 아이콘):
    삭제

    기등록 단어 "조회" (파란색 456EA6 배경 + search 아이콘):
    조회

  6. 예외 및 미매칭 형식 처리
    형식이 맞지 않음 (콜론 1개):
    :abc

    미등록 단어 (일반 kbd 스타일):
