사용자 관리

시스템에 접근하는 모든 사원의 계정을 생성, 수정하고 상태를 관리하는 화면입니다. 본 가이드는 일반 사용자를 위해 공개된 문서이므로 보안 설정은 관리자 페이지를 참고해 주세요.

1. 주요 기능 안내

  • 사원 등록 및 기본 권한(Role) 부여 상태 확인
  • 퇴사자 또는 휴직자 계정 비활성화 (is_active = false) 제어 확인

2. 시스템 계정 상태 정의

상태값 설명
true (Active) 정상 로그인 및 서비스 이용 가능 상태
false (Inactive) 로그인 차단 상태 (Soft Delete 처리 시 변경됨)
