책무 등록하기

  • 신규 책무 등록하기

    1. 등록 버튼을 클릭하여, 책무 등록 팝업을 엽니다.

    2. 책무구분, 책무유형을 차례대로 선택합니다.

    3. 책무를 입력합니다.

    4. 사용시작일을 선택합니다.

    5. 저장 버튼을 클릭하여 저장합니다.


  • 책무 수정하기
    1. 변경하고자 하는 책무의 수정 버튼을 클릭합니다.

  1. 내용을 변경한 후 저장 버튼을 클릭하여 저장합니다.

📌 내용 변경 또는 삭제 시에는 사유를 반드시 입력해야 합니다.


책무 세부내용 등록하기

  1. 등록하고자 하는 책무의 세부내용 등록 버튼을 클릭합니다.

책무유형 등록하기

  • 신규 책무유형 등록하기

    1. 책무유형 관리 버튼을 클릭합니다.

  1. 책무유형 팝업이 열리면, 등록 버튼을 클릭하여 신규 행을 생성합니다.

  2. 신규 행에 책무구분을 선택하고, 책무유형코드와 책무유형명 내용을 입력한 후 저장합니다.


  • 책무유형 삭제하기

    1. 삭제할 책무유형을 선택한 후 삭제 버튼을 클릭합니다.
      ※ 사용하고 있는 책무유형은 삭제할 수 없습니다. (이 경우 '사용' 체크를 해제한 후 저장하세요)

  • 책무유형 수정하기
    1. 수정하고자 하는 책무유형의 내용을 변경한 후 저장하세요
      ※ 책무유형코드는 중복으로 등록될 수 없습니다.

  • 책무유형 작업이 완료되면 닫기 버튼을 클릭하여 팝업을 닫습니다.
