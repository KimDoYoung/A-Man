A-Man (AssetERP 매뉴얼 시스템) 사용자 가이드 (v[version])

본 문서는 A-Man (AssetERP 매뉴얼 시스템)의 문서를 관리하고 생성하는 문서작성자 및 관리자 분들이 시스템의 모든 기능을 200% 활용할 수 있도록 돕기 위해 작성된 상세 도움말 가이드입니다.

1. 개요 및 권한 안내

A-Man은 한국펀드서비스(주)의 핵심 자산운용 패키지인 AssetERP의 도움말과 매뉴얼을 생성하고 관리하는 통합 웹 문서 시스템입니다. 모든 도움말은 직관적이고 표준적인 마크다운(Markdown) 문법을 기반으로 실시간 작성 및 편집됩니다.

사용자의 역할 구분

2. 시스템 관리자 및 문서 작성자 메뉴

관리자 및 작성자 모드로 접속([baseurl]/aman/admin)하면 권한(역할)에 따라 상단 툴바에 다음과 같은 핵심 관리용 메뉴들이 제공됩니다.

2.1 일반 공통 메뉴 (전체 권한)

메뉴명설명
홈으로일반 사용자용 문서 조회 메인 화면(/aman/docs)으로 즉시 이동합니다.
AboutA-Man 매뉴얼 시스템의 릴리즈 정보와 개발 철학, 오픈소스 가이드를 확인합니다.
Help(본 문서) 에디터의 주요 단축키와 스마트 기능을 마스터할 수 있는 도움말 화면을 새 창으로 엽니다.
사용자명 (개인정보 변경)로그인 중인 본인의 이름 버튼이며, 클릭 시 비밀번호와 이메일 등 개인정보를 수정할 수 있습니다.
로그아웃현재 세션을 안전하게 종료하고 로그인 화면으로 이동합니다.

2.2 문서 작성자 전용 메뉴 (일반 작성자)

메뉴명설명
문서 편집왼쪽의 메뉴 트리를 탐색하며 마크다운 에디터를 통해 실시간 매뉴얼을 작성하고 편집하는 메인 대시보드 화면으로 이동합니다.
이미지 편집작성 중인 마크다운 문서 내 이미지를 더블클릭하거나 툴바 버튼을 통해 도형, 텍스트, 화살표 등을 덧그릴 수 있는 이미지 주석 편집기를 토글합니다.
작업이력최근 본인이 생성, 수정 또는 조회했던 문서들의 이력을 조회하고 즉시 해당 문서로 이동할 수 있는 작업 스택 팝업을 띄웁니다 (Alt + H).

2.3 시스템 관리자 전용 메뉴 (Admin 권한)

메뉴명설명
자산 관리문서 작성 시 자주 쓰이는 이모지, 특수기호, 상용구(문구), 템플릿(레이아웃)을 등록하고 편집하여 에디터 툴바 및 팝업 패널에 실시간 연동하는 자산 허브입니다.
메뉴 관리AssetERP의 3단계 메뉴 체계(대분류-중분류-소분류)와 연동되는 트리 카테고리를 추가, 수정, 삭제, 순서 변경하는 화면입니다.
페이지 관리A-Man 시스템 내에 등록된 전체 페이지 목록을 AG-Grid 형태로 조회하며, 배포 상태(작성 중/배포 완료) 필터링, 잠금 상태 강제 해제 등을 지원하는 관리 화면입니다.
사용자 관리문서 작성자 및 관리자 계정을 추가하고, 활성화 여부나 권한(역할) 그룹을 수정/처리하는 화면입니다.
설정시스템 기본 폰트 크기, 이미지 업로드 최대 크기 제한, 링크 타겟 설정 등 시스템 전반에 적용되는 기본 환경 옵션을 설정합니다.
백업 관리시스템 데이터베이스 및 업로드 파일들을 백업 파일로 내보내거나, 과거 시점으로 안전하게 복구할 수 있는 복원 기능을 제공합니다.
로그 조회시스템 내부에서 발생한 주요 에러 및 운영 로그를 실시간에 가깝게 조회하고 분석할 수 있는 화면입니다.

3. 마크다운 에디터 사용법

A-Man 에디터는 마크다운 문법을 신속하고 정확하게 작성할 수 있도록 최첨단 단축키와 반응형 기능을 탑재하고 있습니다.

3.1 특징

  • 미리보기 : 편집과 동시에 미리보기를 통해서 제공되는 문서의 최종형태를 확인할 수 있습니다.
  • 문서 잠금 기능 : 편집하는 동안 문서를 잠금 상태로 놓음으로써 다른 사용자가 편집하는 문서를 변경하지 못하게 할 수 있습니다.
  • 배포상태 변경 : 배포상태가 작성 중인 경우 문서는 일반사용자에게 제공되지 않습니다.
  • 가져오기/내보내기 : 노션의 내보내기 기능으로 다운로드한 zip파일을 가져오기 할 수 있으면 역으로 내보내기를 할 수 있습니다.

3.3 핵심 스마트 편집 기능 사용법

💡 글자 색상 및 배경 색상 순환 변경 (Ctrl+. / Ctrl+/)

  • 글자 색상 순환: 색상을 적용할 텍스트를 드래그 선택한 후 Ctrl + . 키를 입력하면, 누를 때마다 색상이 순환 적용됩니다. (빨강 → 파랑 → 노랑 → 갈색 → 초록 → 색상 제거 순서)
  • 배경 색상(하이라이트) 순환: 강조할 영역을 선택하고 Ctrl + / 키를 입력하면, 배경색이 순환 적용됩니다. (연빨강 → 연파랑 → 연노랑 → 연회색 → 연초록 → 배경색 제거 순서)
  • 에디터 내에서 마우스 우클릭(컨텍스트 메뉴)을 호출하여 원하는 색상을 직접 고를 수도 있습니다.

💡 백틱을 이용한 스마트 버튼 렌더링 및 자동 단축키 태그 적용

  • 백틱을 사용하여 AssetERP에서 사용한 버튼을 표현할 수 있습니다.
  • 버튼의 명칭에 따라서 아이콘과 색상이 결정됩니다.
  • 단 존재하지 않는다면 kbd태그와 같은 형태의 표현이 됩니다.

💡 커서 밀착형 에셋 입력 패널 (Ctrl+1, 2, 3)

  • 마우스 조작 불필요: 본문을 작성하다가 단축키를 누르면, 현재 글자가 적히고 있는 입력 커서 바로 아래에 이모지/기호/상용구 선택 패널이 즉시 나타납니다.

💡 목록 & 인용구 자동 연속 및 자동 종료

  • 글머리 기호( -), 번호 매기기(1.), 인용구(>)를 작성한 뒤 줄 바꿈을 하면, 다음 줄에 자동으로  -, 2., > 기호가 선제적으로 생성됩니다.
  • 만약 목록 작성을 마친 후 일반 줄로 돌아가고 싶다면, 빈 목록 기호 상태에서 Enter를 한 번 더 누릅니다. 그러면 자동으로 줄 시작 기호가 지워지며 목록이 깔끔하게 종료됩니다.

💡 마크다운 표 ↔ CSV 양방향 쾌속 토글 (Ctrl+,)

  • 표 수정이 어려우신가요? 마크다운으로 그려진 복잡한 표를 전체 드래그 선택하고 Ctrl + ,를 누르면, 순식간에 성명,나이,성별 같은 쉼표로 구분된 CSV 데이터 텍스트로 풀어집니다.
  • CSV 상태에서 값을 편하게 수정한 뒤, 다시 드래그하여 Ctrl + ,를 누르면 완벽히 정렬된 깨끗한 마크다운 표로 깔끔하게 조립됩니다.
  • 아무것도 선택하지 않은 채 누르면 3x3의 기본 테이블 템플릿이 삽입됩니다.

💡 클립보드 이미지 직접 붙여넣기 (Ctrl+V)

  • AssetERP 화면을 캡처한 이미지나 PC에서 복사한 이미지 파일을 별도로 저장하고 올릴 필요가 없습니다.
  • 에디터 영역 안에서 단축키 Ctrl + V를 누르면 백엔드 서버에 이미지가 즉시 자동 업로드되고,
    본문에는 ![image](/aman/content/image/...) 와 같은 마크다운 코드가 생성되어 실시간 미리보기에 연동됩니다.

4. 실시간 미리보기 (Live Preview) 및 화면 정렬

  • 실시간 양방향 스크롤 싱크: 에디터 좌측 창을 스크롤하면 우측의 도움말 미리보기 화면도 정확히 같은 위치로 연동되어 움직입니다. 반대로 미리보기를 마우스 휠로 스크롤해도 에디터가 따라 움직입니다.
  • Alt + Z 화면 정렬: 에디터 최하단에서 긴 문서를 편집할 때 모니터 아래쪽에서 계속 고개를 숙이고 작업해야 하는 불편함을 개선했습니다. Alt + Z를 입력하는 즉시 현재 커서 줄이 모니터 화면 정중앙으로 스크롤 재조정됩니다.

6. 고급 기능 가이드

🎨 이미지 편집기 주요 단축키 및 레이어 정렬 가이드

A-Man 이미지 에디터는 마크다운 본문에 삽입된 이미지를 클릭하여 직접 그리기, 텍스트 삽입, 화살표 추가 및 도형 오버레이 등의 주석 작업을 수행할 수 있습니다.

  • 레이어 앞으로 한 단계 조정: Ctrl + ]
  • 레이어 뒤로 한 단계 조정: Ctrl + [
  • 레이어 맨 앞으로 가져오기: Ctrl + Shift + ]
  • 레이어 맨 뒤로 내보내기: Ctrl + Shift + [
