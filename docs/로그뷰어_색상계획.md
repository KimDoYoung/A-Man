# 로그 뷰어 가독성 색상(아이보리/에스프레소) 개선 계획서

이 계획서는 장시간 로그 모니터링 시 눈의 피로를 최소화할 수 있도록, 기존 어두운 터미널 테마를 e-book 스타일의 **아이보리(미색) 배경 + 에스프레소(진한 갈색) 텍스트** 조합으로 변경하는 방안을 설명합니다.

---

## 1. 개요 및 요구사항
- **눈의 피로도 완화**: 대비가 지나치게 강한 검정 배경 대신, 도서 및 전자책 단말기에서 주로 쓰이는 눈이 편안한 미색(Ivory/Sepia) 배경을 적용합니다.
- **가독성 높은 텍스트**: 글자가 눈부시게 밝은 단색 대비를 피하고, 묵직하고 가독성이 높은 고동색/에스프레소 브라운 계열의 텍스트 색상을 기본으로 지정합니다.
- **로그 레벨 식별자 조절**: 배경이 밝은 계열로 전환됨에 따라, 에러/경고/정보 등의 로그 레벨 강조 색상 또한 파스텔/밝은 색에서 어두운 고대비 색상군(크림슨 레드, 딥 앰버, 포레스트 그린)으로 어울리게 재조정합니다.

---

## 2. 변경 계획 세부 내용

### 2.1 [LogPages.tsx](file:///home/kdy987/work/aman/frontend/src/domains/system/LogPages.tsx) 색상 토큰 변경

1. **로그 터미널 컨테이너**:
   - 기존: `bg-slate-950 border-slate-800`
   - 변경: `bg-[#fdfaf2] border-amber-200/60 shadow-inner` (따뜻한 크림/아이보리 톤 배경 및 테두리 적용)

2. **로그 레벨별 텍스트 색상 스키마 (`getLineColorClass`)**:
   - **기본 로그 문자열**: `text-[#4e342e]` (따뜻하고 진한 에스프레소 브라운)
   - **ERROR / Exception / Fail (에러)**: `text-red-700 font-bold` (딥 크림슨)
   - **WARN (경고)**: `text-amber-800 font-semibold` (딥 오렌지/앰버)
   - **INFO (정보)**: `text-emerald-750` -> `text-emerald-700 font-medium` (포레스트 그린)
   - **DEBUG (디버그)**: `text-slate-500` (차분한 그레이)
   - **TRACE (트레이스)**: `text-slate-400` (연한 그레이)

3. **하단 상태 표시 바**:
   - 기존: `border-slate-800/80 text-slate-500`
   - 변경: `border-amber-200/50 text-amber-900/60` (부드러운 하단 구분선 및 텍스트)
