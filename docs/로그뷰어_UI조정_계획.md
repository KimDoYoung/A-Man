# 로그 뷰어 UI 미세 조정 및 보관 로그 목록 토글 기능 추가 계획서

이 계획서는 로그 조회 화면(Log Viewer)의 사용성과 레이아웃을 고도화하기 위한 UI 변경 사항을 기술합니다.

---

## 1. 개요 및 요구사항
- **모니터링 안내 문구 제거**: 터미널 화면 하단에 표시되던 "실시간 모니터링 모드 (로그 파일 tailing)" 텍스트와 아이콘을 제거하여 하단바를 보다 직관적이고 군더더기 없이 슬림하게 만듭니다.
- **보관 로그 파일 목록 토글 기능**: 좌측의 "보관 로그 파일" 목록 패널을 화면에서 접고 펼칠 수 있는 토글 버튼을 제공하여, 필요에 따라 로그 내용 뷰어 창을 넓게 쓸 수 있도록 레이아웃을 유동적으로 제어합니다.
- **폰트 크기 조정**: 로그 가독성을 높이기 위해 텍스트 폰트 크기를 기존 `11px`에서 `12px`(`text-[12px]`)로 1pt 가량 늘려 시각적 편안함을 배가시킵니다.

---

## 2. 변경 계획 세부 내용

### 2.1 [LogPages.tsx](file:///home/kdy987/work/aman/frontend/src/domains/system/LogPages.tsx) UI 컴포넌트 수정

1. **상태값 추가**:
   - `showFileList` boolean 상태 추가 (기본값 `true`)

2. **접이식(토글) 레이아웃 적용**:
   - 좌측 "보관 로그 파일" 칼럼을 `{showFileList && ( ... )}` 조건부 렌더링으로 감싸 노출 상태를 유동화합니다.
   - 콘솔 툴바 영역에 `showFileList` 상태를 제어할 수 있는 토글 버튼 추가:
     - 버튼 텍스트: 접혔을 때 **"보관 로그 보이기"**, 펼쳐졌을 때 **"보관 로그 숨기기"**

3. **하단바 문구 제거**:
   - 콘솔 화면 최하단 좌측의 `<span className="flex items-center gap-1">...</span>` (실시간 모니터링 모드 안내 부분) 코드를 삭제하고 우측의 `총 N개 라인 표시됨` 정렬 값만 중앙 또는 좌측으로 깔끔히 정렬 노출합니다.

4. **로그 줄 폰트 확대**:
   - Terminal 텍스트 렌더러의 className을 `text-[11px]`에서 `text-[12px]`로 수정합니다.
