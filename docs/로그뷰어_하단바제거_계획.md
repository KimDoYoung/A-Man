# 로그 뷰어 하단 정보 영역 제거 계획서

이 계획서는 로그 조회 화면의 콘솔 화면 최하단 분리선 및 통계바 영역을 완전히 제거하여 화면을 한층 더 미니멀하고 깨끗하게 가다듬는 방안을 설명합니다.

---

## 1. 개요 및 요구사항
- **하단 영역 제거**: 콘솔 터미널 내부 하단에 표시되던 `"총 N개 라인 표시됨"` 텍스트와 구분선을 삭제하여 불필요한 공백과 시각적 노이즈를 제거합니다.

---

## 2. 변경 계획 세부 내용

  ```diff
  - </div>
  ```
