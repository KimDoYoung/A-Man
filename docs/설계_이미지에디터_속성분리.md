# 이미지 에디터 속성 독립화 설계 및 접두사(Prefix) 정의서

본 문서는 A-Man 프로젝트의 이미지 에디터 도구에서 각 그리기 아이템(원숫자, 강조 상자, 화살표, 텍스트, 이모지 심볼)이 서로의 스타일 기본값을 공유하여 발생하던 간섭 현상을 해결하기 위한 속성 접두사(Prefix) 명명 규칙 및 적용 설계서입니다.

최종 산출물은 프로젝트 내 도움말을 위한 문서로 관리될 수 있도록 [docs/설계_이미지에디터_속성분리.md](file:///home/kdy987/work/aman/docs/설계_이미지에디터_속성분리.md)로 작성되었습니다.

## 1. 아이템별 속성 접두사(Prefix) 정의

각 그리기 요소는 아래 정의된 독립적인 접두사를 사용하여 상태 변수, 시스템 기본값 및 데이터베이스 JSON 데이터 상의 속성들을 관리하게 됩니다.

| 대상 요소 | 접두사 (Prefix) | 속성 및 설명 | 기존 공용 변수명 (AS-IS) |
| :--- | :--- | :--- | :--- |
| **원숫자**<br>`circle-number` | `circleNumber` | • `circleNumberBgColor`: 원 배경 색상<br>• `circleNumberTextColor`: 원 내부의 숫자 색상<br>• `circleNumberBorderColor`: 원 테두리 색상<br>• `circleNumberBorderWidth`: 원 테두리 두께<br>• `circleNumberFontSize`: 원 내부 숫자의 글자 크기 | • `indigoColor`<br>• `textColor`<br>• `circleBorderColor`<br>• `lineWidth`<br>• `fontSize` |
| **강조 상자**<br>`box` | `box` | • `boxBorderColor`: 강조 상자 테두리 선 색상<br>• `boxLineWidth`: 강조 상자 테두리 선 두께<br>• `boxLineStyle`: 강조 상자 선 종류 (실선/점선)<br>• `boxBgColor`: 강조 상자 내부 배경색<br>• `boxOpacity`: 강조 상자 배경 투명도 (%)<br>• `boxBorderRadius`: 강조 상자 모서리 둥글기 (px) | • `primaryColor`<br>• `lineWidth`<br>• `boxLineStyle`<br>• `boxBgColor`<br>• `boxOpacity`<br>• `boxBorderRadius` |
| **화살표**<br>`arrow` | `arrow` | • `arrowColor`: 화살표 선/머리 색상<br>• `arrowLineWidth`: 화살표 선 두께<br>• `arrowLineStyle`: 화살표 선 종류 (실선/점선) | • `primaryColor`<br>• `lineWidth`<br>• `boxLineStyle` |
| **일반 텍스트**<br>`text` | `text` | • `textTextColor`: 일반 텍스트 글자 색상<br>• `textFontSize`: 일반 텍스트 글자 크기 | • `textColor`<br>• `fontSize` |
| **이모지 심볼**<br>`symbol` | `symbol` | • `symbolEmoji`: 이모지 심볼 문자 (예: 💡, ⚠️)<br>• `symbolScale`: 이모지 심볼 크기 배율 (1 ~ 5단계) | • `selectedEmoji`<br>• `symbolScale` |

## 2. 하위 호환성 (Backward Compatibility) 지원 설계

이전에 작성되어 SQLite 데이터베이스에 보관 중인 `jsonData` 및 이전 개인화 `user-settings`를 불러올 때, 새로운 독립 변수 명명법으로 무중단 복원이 가능하도록 Fallback 체계를 적용합니다.

### 2.1 JSON 데이터(`jsonData`) 로드 시 Fallback 우선순위
이전 레코드 로딩 시 아래의 우선순위로 값을 할당받아 에디터의 State를 복원합니다.

1. **원숫자 속성 복원**
   - 배경색: `data.circleNumberBgColor` → `data.indigoColor` → 시스템 기본값
   - 텍스트색: `data.circleNumberTextColor` → `data.textColor` → 시스템 기본값
   - 테두리색: `data.circleNumberBorderColor` → `data.circleBorderColor` → 시스템 기본값
   - 테두리두께: `data.circleNumberBorderWidth` → `data.lineWidth` → 시스템 기본값
   - 폰트크기: `data.circleNumberFontSize` → `data.fontSize` → 시스템 기본값

2. **강조 상자 속성 복원**
   - 테두리색: `data.boxBorderColor` → `data.primaryColor` → 시스템 기본값
   - 선두께: `data.boxLineWidth` → `data.lineWidth` → 시스템 기본값
   - 선종류: `data.boxLineStyle` → 시스템 기본값
   - 배경색: `data.boxBgColor` → 시스템 기본값
   - 투명도: `data.boxOpacity` → 시스템 기본값
   - 모서리 둥글기: `data.boxBorderRadius` → 시스템 기본값

3. **화살표 속성 복원**
   - 선색상: `data.arrowColor` → `data.primaryColor` → 시스템 기본값
   - 선두께: `data.arrowLineWidth` → `data.lineWidth` → 시스템 기본값
   - 선종류: `data.arrowLineStyle` → `data.boxLineStyle` → 시스템 기본값

4. **텍스트 속성 복원**
   - 글자색: `data.textTextColor` → `data.textColor` → 시스템 기본값
   - 글자크기: `data.textFontSize` → `data.fontSize` → 시스템 기본값

5. **이모지 심볼 속성 복원**
   - 이모지: `data.symbolEmoji` → `data.selectedEmoji` → 시스템 기본값
   - 크기배율: `data.symbolScale` → 시스템 기본값

### 2.2 사용자 개인 설정 (`user-settings`) 조회 시 Fallback
서버에서 기존 설정 키(`image_editor_defaults_user_{userId}`)로 조회한 JSON 객체를 파싱할 때도 동일한 Fallback 규칙을 통해 상태 변수로 매핑합니다.

## 3. 리팩토링 수행 파일 및 영향도
* **영향 범위**: 이미지 에디터 단독
* **수정 대상 파일**:
  1. `frontend/src/components/imageditor/image_items_defaults.ts`: 접두사 적용된 독립 기본값 상수로 재구성
  2. `frontend/src/components/imageditor/FloatingPropertyPanel.tsx`: 새로운 Props 명세에 맞춘 바인딩 및 UI 노출 처리
  3. `frontend/src/components/imageditor/ActionImageEditor.tsx`: 상태 변수 교체, 렌더링 루프 바인딩 교체, 저장 및 복원 로직 교체
